# Compiling
I use Visual Studio 2019 to compile this project. The following libraries are used:
- SDL version 2.0.18
- SDL_image 2.0.5
- SDL_ttf 2.0.15 (currently unused but still included)
# Current Core Features
- Variable jumping Mario
- Camera system
- Parallaxing backgrounds
